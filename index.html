Welcome to DitDah, the world's leading provider of Morse Code translation services!

For English to Morse Code:

Type message in the first text area. Do not include any punctuation, then press the encode button. Your message will appear in the second field after translation.

Morse Code to English:

Enter message text area and enter a message using dit (.) and dah (-). Letters must be separated by a space, and words must be separated by '//'. Press decode and the English translation will appear in the top text area.

Click arrow to begin!

Type English in the box below with no punctuation. Click 'ENCODE' to translate!
